# SERENE Asset Guide

Place all image and video files in `public/img/` (and `public/videos/` for video).  
Paths below match the data files in `data/` — drop files in with these exact names and no code changes are needed.

---

| File | Path | Recommended size | Notes |
|------|------|------------------|-------|

## Image guidelines

- Format: JPG for photos, SVG for logo/signature
- Optimize before adding (target &lt; 300 KB per photo where possible)
- Every image should have a descriptive alt text already defined in `data/`
- Use high-quality, cohesive photography — warm tones, natural light, minimal clutter
